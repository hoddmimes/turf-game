TurfGame Extra

Nedan finns länkar till ett några små exempelprogram som alla använder Turf API:et. Tanken är att programmen kommer att få leva tillsvidare men med sedvanlig "garantier" d.v.s. "AS-IS".


Zon "takeover" prenumerationer

Tjänsten tillåter att du kan sätta upp prenumerationer på när dina favorite zoner bli tagna. När det sker så skickas ett mail till dig. Denna tjänst kräver att du registrear dig. Detta för att användare inte ska kunna sätta upp prenumeration för icke auktoriserade mail konton.
Du kommer åt denna tjänst via länken här


Dagspoäng för regioner

Här visas en lista över vilka Turfare som tagit mest poäng i olika regioner under det senaste dygnet. Listan uppdateras varje minut och nollställs mid midnatt. För närvarande visas statistik för de 50 mest aktiva turfarna för några av de större turfmetropolerna i Sverige. Om någon mot förmodan skulle vilja få statistik för ytterligare region, så kan det nog ombesörjas om ni mail till undertecknad.
Du kommer åt denna tjänst via länken här


Regionsstatistik

Här visas en statistik jämförelse mellan olika regioner både löpande och de senaste 48 timmarna. Statistiken uppdateras löpande var 30:e sekund. Här kan typiskt se hur genomsnittsvärden för tid och distans mellan två stycken takes. PPH och TP per take, etc.
Du kommer åt denna tjänst via länken här


  • Regionsstatistik
  • Daglig toplista per region
  • Zon "takeover" prenumerationer
